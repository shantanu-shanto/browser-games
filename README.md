## How to Play

- **Arrow Left/Right**: Move your car
- **Arrow Up/Down**: Control speed (30-150 km/h)
- **Avoid**: Obstacle cars
- **Collect**: Gold coins for bonus points

## Technologies Used

- Three.js (3D graphics library)
- HTML5
- CSS3
- Vanilla JavaScript

## Installation

1. Clone this repository
2. Open `index.html` in your browser
3. Start playing!

No build process or dependencies required - runs directly in the browser!
